9. http의 특징 : http 텍스트를 교환 하는 것으로
-> http -> hyper text transport protocol -> hyper text -> 인터넷에서 전송 되어지는 데이터 -> 교환 하는거 -> 응용계층 -> 인터넷에서 이 텍스트를 남이 가로채서 엿볼 수 있다.(snapping)
